# Wealth OS — Personal Daily Operating System (Andile)

**Core Philosophy**  
Systems > Motivation. Delayed gratification as daily practice. Batch everything. Protect deep work. Tie every action to attention → trust → email list → recurring revenue.

## Daily Operating Rhythm (Recommended)

| Time Block       | Focus                          | Key Actions                                                                 | Tools                  |
|------------------|--------------------------------|----------------------------------------------------------------------------|------------------------|
| 05:30–06:30     | Morning OS Review             | Review yesterday’s numbers, top 3 priorities, delayed gratification score   | Notion Wealth OS      |
| 06:30–08:00     | Deep Work / Content Creation  | Film or write flagship piece (YouTube or long-form)                         | Batch filming setup   |
| 08:00–09:00     | Content Processing            | Edit/Reels + captions + distribution scheduling                             | CapCut + Notion       |
| 09:00–10:30     | Platform & Product Work       | Monorepo work, bug fixes, feature development                               | Cursor / VS Code      |
| 10:30–11:00     | Movement + Reset              | Walk, sunlight, breathing trainer                                           | —                     |
| 11:00–12:30     | Revenue & Outreach            | B2B messages, email sequences, offer delivery                               | Notion + Gmail        |
| 12:30–13:30     | Lunch + Family                | —                                                                         | —                     |
| 13:30–15:30     | Second Deep Work Block        | Academy content, playbook writing, or high-leverage project                 | —                     |
| 15:30–16:30     | Admin & Systems               | Finance review, budgeting update, tool maintenance                          | SA Budgeting tool     |
| 16:30–17:30     | Content Distribution          | Post scheduled content, engage on LinkedIn/X, reply to comments             | Buffer / Native       |
| Evening (Sun)   | Weekly Review                 | Score week, adjust priorities, plan next week’s batch                       | Notion                |

## Weekly Non-Negotiables
- 1× long-form YouTube (18–22 min)
- 3–5 Reels
- 2–3 LinkedIn posts
- 1 Wealth OS / Academy email to list
- Revenue & pipeline review (every Sunday)
- Delayed gratification “streak” check

## Decision Framework
Before saying yes to anything: Does this move attention, trust, email list, or recurring revenue? If not in top 3 priorities this week — defer or delegate.
